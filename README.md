
# Netflix Interface Recreation

Esse projeto foi desenvolvido para fins de estudo e prática de desenvolvimento front-end, utilizando como referência a interface da Netflix.

## 📎 Live Demo
https://biancassp.github.io/projeto-netflix/

## 📌 Sobre o Projeto
A página principal apresenta uma estrutura inspirada em plataformas de streaming com um **carrosel de imagens interativo**.

Ao clicar em "Continuar", o usuário é levado para uma tela de transição que simula a revelação de um novo lançamento. As informações são exibidas gradualmente e, ao final da sequência,
a aplicação retorna automaticamente para a tela principal.

Durante o desenvolvimento, foram trabalhados principalmente conceitos de manipulação do DOM, controle de tempo em animações e organização da interface.

## 🚀 Tecnologias Utilizadas
* **HTML5** para estruturar a página e organizar os elementos da interface;
* **CSS3** para estilização e layout;
* **JavaScript** para criar as interações da aplicação;
* **Swiper.js** para implementar o carrossel de imagens.
